Sudoku

This was created for our Degree Project in Computer Science, first cycle. It contains many different versions of algorithms for solving sudoku puzzles (of chosen size, although the larger ones are very slow). Currently, they are all based on backtracking, with different additions for speeding up.

Abbreviations

Classnames use abbreviations to let you know what they use:

  • BT: BackTracking
  • FC: Forward Checking
  • CP: Constraint Propagation
  • MRV: Minimum Remaining Value
